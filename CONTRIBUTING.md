# Berkontribusi

Terima kasih atas minat Anda pada **valewin**.

## Cara berkontribusi

1. Fork repositori ini
2. Buat branch fitur (git checkout -b fitur/peningkatan)
3. Commit perubahan dengan pesan yang jelas
4. Buka Pull Request

## Pedoman

- Ikuti [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)
- Perubahan konten Jekyll: uji lokal dengan bundle exec jekyll serve bila memungkinkan
- Diskusi awal bisa lewat tab **Discussions**

## Sponsorship

Tombol **Sponsor** di repositori mengarah ke profil maintainer dan portal resmi di [.github/FUNDING.yml](.github/FUNDING.yml) (https://valewin.org).
